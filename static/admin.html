🗑️ 清空向量库
删除 Milvus 中的全部向量数据（Collection 将被彻底删除）。
此操作不可撤销，清空后需重新导入数据才能恢复问答功能。
⚠️ 清空后，所有已索引的文档片段将永久丢失。请确保已备份原始数据文件，或已准备好重建所需的文档。

⚠️ 重建过程中，现有向量库将被先清空再重新写入，耗时取决于文档量与 Embedding API 速度，请勿关闭页面。
